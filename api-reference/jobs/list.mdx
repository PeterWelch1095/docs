The List Jobs endpoint returns job postings visible to your institution on Handshake, including full-time roles, internships, part-time jobs, and co-ops. Use pagination and filtering parameters to work through large result sets or to sync only records that have changed since a previous request.

## Endpoint

```
GET https://edu-api.joinhandshake.com/v1/jobs
```

## Authentication

Include your API key in the request header:

```
x-api-key: <your-api-key>
```

## Query parameters

| Parameter | Type | Description |
|---|---|---|
| `page_size` | integer | Number of results per page. Maximum `100`. |
| `page_cursor` | string | Cursor returned in a previous response as `next_cursor`. Pass this value to retrieve the next page. |
| `updated_since` | string (ISO 8601) | Return only records updated at or after this timestamp. Example: `2024-01-15T00:00:00Z`. |
| `sorting_column` | string | Field to sort results by. Accepted values: `id`, `updated_at`, `created_at`, `title`. |
| `sorting_order` | string | Sort direction. Accepted values: `asc`, `desc`. |

## Response

| Field | Type | Description |
|---|---|---|
| `results` | array | Array of job objects for the current page. |
| `next_cursor` | string \| null | Cursor for the next page of results. `null` when there are no further pages. |
| `total_count` | integer | Total number of job records matching the query. |

### Job object fields

<Expandable title="Job object">

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique Handshake identifier for the job. |
| `title` | string | Job title as entered by the employer. |
| `employer_id` | string | Unique identifier of the employer who posted the job. |
| `employer_name` | string | Display name of the employer. |
| `job_type` | string | Type of role. One of: `full_time`, `internship`, `part_time`, `co_op`. |
| `status` | string | Current posting status. One of: `active`, `expired`, `draft`. |
| `description` | string | Full job description text. |
| `created_at` | string (ISO 8601) | Timestamp when the job was created. |
| `updated_at` | string (ISO 8601) | Timestamp when the job was last updated. |

</Expandable>

## Example request

```bash
curl --request GET \
  --url "https://edu-api.joinhandshake.com/v1/jobs?page_size=25&sorting_column=updated_at&sorting_order=desc" \
  --header "x-api-key: <your-api-key>"
```
