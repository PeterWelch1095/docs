Student labels are custom tags applied to student profiles in Handshake for segmentation. Your institution's administrators can create labels to group students by cohort, program, scholarship status, or any other category. The student labels endpoint returns individual label assignment records that you can join to student records using the `student_id` field.

```
GET https://edu-api.joinhandshake.com/v1/student_labels
```

## Query Parameters

<ParamField query="page_size" type="integer">
  Number of label assignment records to return per page. Maximum value is `100`. Defaults to `25`.
</ParamField>

<ParamField query="page_cursor" type="string">
  Opaque pagination cursor. Pass the `next_cursor` value from the previous response to retrieve the next page.
</ParamField>

<ParamField query="updated_since" type="string">
  ISO 8601 datetime (e.g. `2024-01-15T00:00:00Z`). When provided, only label assignments created or updated at or after this timestamp are returned.
</ParamField>

<ParamField query="sorting_column" type="string">
  Field to sort results by. Accepted values: `id`, `updated_at`, `created_at`.
</ParamField>

<ParamField query="sorting_order" type="string">
  Sort direction. Accepted values: `asc` (ascending) or `desc` (descending).
</ParamField>

## Response Fields

<ResponseField name="results" type="array">
  Array of label assignment objects. Each object links a single label to a single student.

  <Expandable title="Label assignment object">
    <ResponseField name="id" type="string">
      Unique identifier for this label assignment record.
    </ResponseField>

    <ResponseField name="student_id" type="string">
      Handshake ID of the student this label is assigned to. Use this to join with records from the [List Students](/api-reference/students/list) endpoint.
    </ResponseField>

    <ResponseField name="label_id" type="string">
      Unique identifier of the label itself. The same `label_id` appears on every assignment for a given label.
    </ResponseField>

    <ResponseField name="label_name" type="string">
      Display name of the label as configured in your Handshake account (e.g. `"Honors Program"` or `"First-Generation"`).
    </ResponseField>

    <ResponseField name="created_at" type="string">
      ISO 8601 datetime when this label was assigned to the student.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="next_cursor" type="string | null">
  Cursor to pass as `page_cursor` on the next request to retrieve the following page. Returns `null` when you have reached the last page.
</ResponseField>

<ResponseField name="total_count" type="integer">
  Total number of label assignment records matching the query, across all pages.
</ResponseField>

## Example Request

```bash
curl "https://edu-api.joinhandshake.com/v1/student_labels?page_size=3&sorting_column=created_at&sorting_order=desc" \
  -H "x-api-key: YOUR_API_KEY"
```
