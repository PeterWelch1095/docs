Retrieve full details for a single career fair using its Handshake career fair ID. Use this endpoint when you need the complete record for a specific fair, such as confirming dates and format before fetching related session or attendance data.

## Endpoint

```
GET https://edu-api.joinhandshake.com/v1/career_fairs/{id}
```

## Authentication

Include your API key in the request header:

```
x-api-key: <your-api-key>
```

## Path parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `id` | string | Yes | The Handshake career fair ID. |

## Response fields

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique Handshake identifier for the career fair. |
| `name` | string | Name of the career fair. |
| `start_at` | string (ISO 8601) | Scheduled start time of the career fair. |
| `end_at` | string (ISO 8601) | Scheduled end time of the career fair. |
| `virtual` | boolean | `true` if the career fair is held online; `false` if in-person. |
| `created_at` | string (ISO 8601) | Timestamp when the career fair record was created. |
| `updated_at` | string (ISO 8601) | Timestamp when the career fair record was last updated. |

## Example request

```bash
curl --request GET \
  --url "https://edu-api.joinhandshake.com/v1/career_fairs/10044" \
  --header "x-api-key: <your-api-key>"
```
