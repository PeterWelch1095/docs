The List Career Fairs endpoint returns career fair records organized at your institution on Handshake. Sorting by `start_at` lets you retrieve fairs in chronological order, while `updated_since` supports incremental sync of recently modified records.

## Endpoint

```
GET https://edu-api.joinhandshake.com/v1/career_fairs
```

## Authentication

Include your API key in the request header:

```
x-api-key: <your-api-key>
```

## Query parameters

| Parameter | Type | Description |
|---|---|---|
| `page_size` | integer | Number of results per page. Maximum `100`. |
| `page_cursor` | string | Cursor returned in a previous response as `next_cursor`. Pass this value to retrieve the next page. |
| `updated_since` | string (ISO 8601) | Return only records updated at or after this timestamp. Example: `2024-01-15T00:00:00Z`. |
| `sorting_column` | string | Field to sort results by. Accepted values: `id`, `updated_at`, `start_at`. |
| `sorting_order` | string | Sort direction. Accepted values: `asc`, `desc`. |

## Response

| Field | Type | Description |
|---|---|---|
| `results` | array | Array of career fair objects for the current page. |
| `next_cursor` | string \| null | Cursor for the next page of results. `null` when there are no further pages. |
| `total_count` | integer | Total number of career fair records matching the query. |

### Career fair object fields

<Expandable title="Career fair object">

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique Handshake identifier for the career fair. |
| `name` | string | Name of the career fair. |
| `start_at` | string (ISO 8601) | Scheduled start time of the career fair. |
| `end_at` | string (ISO 8601) | Scheduled end time of the career fair. |
| `virtual` | boolean | `true` if the career fair is held online; `false` if in-person. |
| `created_at` | string (ISO 8601) | Timestamp when the career fair record was created. |
| `updated_at` | string (ISO 8601) | Timestamp when the career fair record was last updated. |

</Expandable>

## Example request

```bash
curl --request GET \
  --url "https://edu-api.joinhandshake.com/v1/career_fairs?page_size=25&sorting_column=start_at&sorting_order=asc" \
  --header "x-api-key: <your-api-key>"
```
